# 3_department.py 명세서

## 목적

부서 및 직급별 사용 현황을 분석하는 페이지.
드롭박스 선택값에 따라 부서별/직급별 차트 데이터가 변경됨.

---

## 기본 규칙

- 데이터는 st.session_state에서만 가져옴. 직접 로드 금지.
- merge, 매핑, 컬럼 처리 금지. 데이터 가공은 data.py에서 완료된 상태.
- config.py의 상수값 사용. 연도/날짜 하드코딩 금지.
- st.session_state에서 가져오는 키: df_users, df_login, df_download, df_proposal, date_preset, date_range, sel_dept, sel_rank

---
## 부서 기준 특이사항

이 페이지의 부서기준은 다른 페이지와 다름
다른 페이지: df_users의 "부서" 컬럼 사용 (부서명 -> 본부/실 -> 사업부 fallback)
이 페이지: 사업부 컬럼 기준으로 그룹핑. 단 아래 예외 있음.
예외 처리:
- config.DEPT_SHOW_AS_HQ에 포함된 본부/실 값은 사업부 대신 본부/실 값으로 표시
- config.py에 추가할 상수: DEPT_SHOW_AS_HQ = ["CP실", "주최사업실"]
df_users의 각 행에 대해:
  - 본부/실 값이 config.DEPT_SHOW_AS_HQ에 포함되면 본부/실 값 사용
  - 그 외는 사업부 값 사용
  - 사업부 값도 없으면 해당 행 차트에서 제외

## 페이지 내부 필터: 로그 종류 드롭박스

위치: 제목 바로 아래 행 우측 (st.columns([4,1]) 오른쪽)
선택지: 로그인 / 제안서 / 다운로드 전체 / 프로젝트 찾기 / 운영자료 찾기 / 서포트센터
적용 범위: 섹션1(부서별 차트), 섹션2(직급별 차트)만 변경. 섹션3(인원표)은 변경 없음.

드롭박스 선택값에 따른 데이터 분기:
- 로그인: f_login 사용
- 제안서: f_proposal 사용
- 다운로드 전체: f_download, f_proposal 전체 사용
- 프로젝트 찾기: f_download에서 경로 메뉴명에 "프로젝트" 포함된 행만
- 운영자료 찾기: f_download에서 경로 메뉴명에 "운영자료" 포함된 행만
- 서포트센터: f_download에서 경로 메뉴명에 "서포트" 포함된 행만

## 직급 그룹 분류 기준

차트에서 대리/사원 vs 차장이상 구분 시 아래 기준 사용.

- 대리/사원 그룹: 직급그룹 컬럼 값이 '실무자(사원/대리)'
- 차장이상 그룹: 직급그룹 컬럼 값이 '관리자(차장↑)'
- 임원 그룹 : 직급그룹 컬럼 값이 '임원'

3개 그룹 모두 별도 선/막대로 표시
직급그룹 컬럼은 df_login, df_download, df_proposal에 이미 조인되어 있음.

## 섹션 2: 직급별 사용 현황 (2행 1열)

### 막대그래프
- 날짜필터 : 사이드바의 공통 필터 사용
- x축: 임원을 제외한 직급 전체
- y축: 횟수

- 드롭박스 선택 기준 데이터 변경

---

## 섹션 3: 부서/직급별 인원 (3행 2열)

### 좌측: 표
- 출처: df_users (raw 직원현황, 전체 필터 미적용)
- 부서별 직급 인원 카운팅
- 컬럼: 부서 / 직급 / 인원수
- 부서 컬럼: df_users의 "부서" 컬럼 사용
- 직급 컬럼: config.YEAR_COL_RANK.format(year=config.CURRENT_YEAR) 사용
- 계산: groupby(['부서', 직급컬럼]).size().reset_index(name='인원수')

### 우측: 직급별 합계
- 출처: 드롭박스 선택 기준 데이터 (섹션1, 2와 동일하게 변경)
- 직급그룹별 활동 집계
  - 차장이상 그룹: 직급그룹 == '관리자(차장↑)' 횟수
  - 대리/사원 그룹: 직급그룹 == '실무자(사원/대리)' 횟수
- 사용률 계산: 각 그룹 횟수 / (차장이상 횟수 + 대리사원 횟수) * 100
- 임원 그룹은 집계에서 제외
- 마우스오버 시 인원수 + % 동시 표시
- 드롭박스 선택 기준 데이터 변경

## config.py 상수 참조

```
config.CURRENT_YEAR         현재 연도 (2026)
config.YEAR_COL_DEPT        "{year}_부서명"
config.YEAR_COL_RANK        "{year}_통계 직급"
config.COL_NAME_EMAIL       "PRS ID"
```

df_login, df_download, df_proposal에 조인된 주요 컬럼:
- 부서: 조인 후 "부서" 컬럼으로 통일
- 직급그룹: 실무자(사원/대리) / 관리자(차장↑) / 기타
